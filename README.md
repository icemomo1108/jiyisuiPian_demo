# 记忆碎片 DEMO 试玩版

## 本地运行步骤

1. 安装依赖：
   ```bash
   npm install
   ```

2. 运行项目：
   ```bash
   npm run dev
   ```

3. 打开浏览器访问：
   ```
   http://localhost:5173
   ```

## 部署到 Vercel

1. 上传整个项目到 GitHub 仓库或直接上传到 Vercel
2. 在项目环境变量中配置你的 OpenAI API Key：
   ```
   VITE_OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxx
   ```

3. 完成部署后即可在线试玩。
